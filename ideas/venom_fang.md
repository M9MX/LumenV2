# Venom Fang

**Source Pack:** Custom Design (Poison/Serpent Theme)

## Base Item
- Material: `DIAMOND_SWORD`
- Model: `lumen:venom_fang`
- Display Name: `<green><bold>Venom Fang`
- Base Damage: Default
- Base Attack Speed: Default
- Unbreakable: Yes
- Static model (no progression variants)

## Abilities

### Right-Click: Venom Cloud
- **Cloud Duration:** 20 seconds
- **Poison Duration on Enter:** 10 seconds (reapplies if stay in or re-enter)
- **True Damage:** 1 HP/second to all enemies inside (armor-ignored, separate effect)
- **Player Immunity:** Player does not get poisoned by own cloud
- **Cooldown:** 60 seconds

## Passive: Venom Conversion
- **Trigger:** Hold weapon while afflicted with poison effect
- **Effect:** Converts poison effect to Regeneration
- **Duration & Level:** Same as original poison effect
- **Application:** Encourages weapon holding and risk/reward gameplay

## Theme
- Poison/serpent-inspired
- Green and dark green color scheme
- Toxic, venomous aesthetic
- Reflects poison mastery and conversion mechanics
