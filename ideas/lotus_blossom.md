# Lotus Blossom

**Source Pack:** Custom Design (Eastern Floral Theme)

## Base Item
- Material: `DIAMOND_SWORD`
- Model: `lumen:lotus_blossom`
- Display Name: `<light_purple><bold>Lotus Blossom`
- Base Damage: Default
- Base Attack Speed: Default
- Unbreakable: Yes
- Static model (no progression variants)

## Abilities

### Right-Click: Blossom Shield
- **Duration:** 10 seconds
- **Effect:** Spawns 5 rotating spore blossom block displays around player (pink part automatically facing outward)
- **Damage Reduction:** Halves damage taken
- **Thorns Effect:** Enemy takes 1.5x the damage they dealt to you
- **Cooldown:** 60 seconds
